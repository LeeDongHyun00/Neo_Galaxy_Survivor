# 🚀 Neon Galaxy Survivor V7

현대적인 TypeScript + Vite 아키텍처로 재구축된 로그라이트 우주 슈팅 게임입니다.

![Version](https://img.shields.io/badge/version-7.0.0-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-5.3-blue)
![Vite](https://img.shields.io/badge/Vite-5.0-purple)

## ✨ 주요 기능

### 게임플레이

- 🎮 **자동 발사 시스템** - Vampire Survivors 스타일
- 👾 **다양한 적 타입** - Basic, Shooter, Exploder, Tank, Boss
- ⚡ **7가지 엘리트 능력** - 드론, 해킹, 미사일, 플라즈마 필드 등
- 📈 **레벨업 & 업그레이드** - 6가지 일반 업그레이드
- 🌊 **무한 웨이브** - 5웨이브마다 보스 등장

### 시스템

- 📊 **로컬 리더보드** - 상위 10개 기록 저장
- 💾 **진행도 저장** - localStorage 기반 영구 저장
- 🎨 **향상된 비주얼** - 파티클 효과, 화면 흔들림
- 🔊 **개선된 사운드** - Web Audio API 활용
- 📱 **최적화된 모바일** - 빠른 조준, 작은 적 크기

### 기술 스택

- **TypeScript** - 타입 안전성
- **Vite** - 빠른 개발 서버 & 빌드
- **Canvas API** - 2D 렌더링
- **Web Audio API** - 사운드 효과
- **LocalStorage** - 데이터 저장

## 🎯 업그레이드 목록

### 일반 업그레이드

- ☄️ **멀티 샷** - 발사체 +1
- ⚡ **광속 장전** - 공격 속도 +15%
- 💥 **화력 증강** - 데미지 증가
- 🚀 **부스터** - 이동 속도 증가
- ❤️ **선체 수리** - 체력 50% 회복
- 🛡️ **쉴드 확장** - 최대 체력 증가

### 엘리트 업그레이드 (보스 처치 시)

- 🛰️ **윙맨** - 공격 드론 소환
- 🧠 **브레인 잭** - 주기적으로 적 해킹
- ⚡ **플라즈마 필드** - 주변 지속 피해
- 🚀 **스마트 미사일** - 유도 미사일 발사
- 💉 **나노 머신** - 체력 자동 회복
- 💣 **연쇄 반응** - 적 처치 시 폭발
- ⏳ **타임 시프트** - 적 탄환 감속
